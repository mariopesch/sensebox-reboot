openSenseMap

Die openSenseMap ist unsere Web-Plattform für die offenen Daten der senseBox und anderer Geräte. Jede senseBox:home übeträgt ihre Messdaten direkt auf diese Karte, wo sie betrachtet, analysiert und heruntergeladen werden können. Auch mit der senseBox:edu erfasste Daten können auf der openSenseMap angezeigt werden.

Alle auf der openSenseMap veröffentlichten Daten sind Open Data unter der Public Domain Dedication and License 1.0.

API

Mit unserer offenen API lassen sich sämtliche Sensordaten für andere Anwedungen weiterverwenden. Außerdem können Sensordaten von alternativen Quellen in die openSenseMap eingebunden werden.
